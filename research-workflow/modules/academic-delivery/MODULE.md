# academic-delivery

## 目标

将研究中间产物整理为可对外使用的交付物，并通过质量门禁。

## 执行规范

1. 先推荐 1-2 个相关 Skills（如 scientific-writing, docx/pptx/pdf）。
2. 统一结构：目标、方法、证据、发现、限制、建议。
3. 统一术语、引用与图表说明。
4. 输出至 `deliverables/`（保留源文件，不覆盖原稿）。
5. 执行质量门禁并给出评分。

## 交付清单建议

- Executive Summary
- Full Report
- Evidence Table
- Next-step Recommendations
