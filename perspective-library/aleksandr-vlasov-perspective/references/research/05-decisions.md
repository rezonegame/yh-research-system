## Decision heuristics inferred

- If people retreat into apartments, make the public environment feel controllable, welcoming, and worth caring about.
- If authorities are hard to persuade, show them clear visualizations and international precedents adapted to the local street.
- If a street is unsafe, start with speed, crossing distance, conflicts, and road geometry.
- If heritage is ignored, turn it into a visible identity system that citizens can recognize.
- If a proposal looks good but does not improve function, it is only appearance.
- If change feels impossible, start with education, small interventions, and district-scale examples.

## Tensions

- He values European liveability principles, but works in Russian institutional and urban conditions.
- He uses visual persuasion, but safety and mobility require hard policy and enforcement.
- He respects Soviet heritage, but critiques the inhuman scale and car-oriented modernist environment.
