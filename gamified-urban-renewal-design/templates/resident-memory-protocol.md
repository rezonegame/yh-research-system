## 3. Consent Levels

| Level | Meaning | Allowed use |
|---|---|---|
| Private note | Not quoted or displayed | Internal design understanding only |
| Anonymous quote | No name or identifying details | Report / workshop synthesis |
| Public display | Can be shown in route/exhibition | Public materials with review |
| Design requirement | Memory becomes a planning constraint | Translation matrix / masterplan |
| No use | Withdrawn or sensitive | Do not use |

## 4. Interview Questions

1. What is your earliest memory of this place?
2. Which route, courtyard, shop, tree, sound, smell, or daily scene matters to you?
3. What changed for the better?
4. What was lost or made harder?
5. Which memories are suitable for visitors to know?
6. Which memories should not be commercialized or displayed?
7. What visitor behaviour would feel respectful?
8. What would make residents benefit from renewal?
9. What should the game/route never ask people to do?
10. What is one improvement you want to see in the next 3 months?

## 6. Transformation Rules

| Memory type | Can become | Do not do |
|---|---|---|
| Everyday routine | Route prompt, design requirement | Romanticize inconvenience |
| Local business memory | Merchant story, product route | Force merchant participation |
| Heritage memory | Interpretation, quiet observation | Fictionalize or gamify sacred/sensitive content |
| Relocation/conflict memory | Governance issue, private synthesis | Public spectacle or identifiable quote |
| Child/youth memory | Play route, safety design | Collect identifiable data without guardian consent |

## 7. Review

Before public use, review with the interviewee or community representative, project operator, heritage/cultural advisor when relevant, and privacy/legal reviewer when identifiable material appears.
